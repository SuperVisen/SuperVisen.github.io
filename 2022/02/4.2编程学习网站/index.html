编程自学指南

发表于2022-02-16|更新于2022-05-12|疯狂安利
|阅读量:

引子

&nbsp;&nbsp;&nbsp;&nbsp;👋这里是Visen.蛮多人问我如何自学编程，这里提供一篇文章，点出自身学习编程的步骤，也希望大家给予宝贵意见！

我的学习思路是

  1. 计算机基础：从计算机导论开始，对整体有把握！然后学习计算机学科的基础知识，补齐专业知识
  2. 学习路线：了解想要从事方向的学习路线
  3. 分类学习：对自己方向进行分类学习，按照学习路线完善自己技术栈
  4. 掌握工具：掌握必备工具，比如Git、Github等

综合网站

CS-WIKI：记录博主的学习之路，还是很贴合实际的，博主还总结后开发了Leetcode-Offer刷题网站！实在大佬。类似的还有菜饼同学

菜鸟教程：站长Runoob一个人维护的网站，将大家的知识整理成笔记。导航栏的在线菜鸟工具也很不错。类似的还有W3school/W3Cschool/C语言中文网

自学计算机科学：对 TeachYourselfCS 内容的中文翻译，可以借助建议的教材或者视频课程来自学计算机科学

廖雪峰的网站：廖雪峰的Git教程是必须看看的！

编程学院：简单注册后，可以左侧侧边栏展示教程，配合右侧在线Vscode编程，一步步教学，免费版用户的使用时长就够用了

Coursera：与‎‎200多所领先的大学‎‎和公司合作，为全球的个人和组织提供在线学习，这里的一些计算机课程很推荐，比如北大的计算机组成，斯坦福大学的机器学习等

编程自学之路：整理收录几个大方向（岗位）的学习路线和知识点大梳理

帅地玩编程：面向工作，更新一些学习资源和大厂面经

脚本之家：专业的收藏整理多类脚本学习资料的个人网站，可以搜索书籍和教程

C语言网：网站更新的内容以C、C++语言来玩单片机的教程，简单实用

PHP中文网：不只是PHP学习，里面还有很多其他语言教程和在线工具

Markerhub：梳理Java知识、解析开源项目，内容以项目为主

计算机基础

数学基础：《线性代数》、《离散数学及其应用》

计算机入门知识：《计算机科学导论》

计算机组成：计算机组成 北大

计算机网络：计算机网络微课堂

操作系统

数据结构及算法：

  • 《大话数据结构》
  • 《图解算法》
  • CS-Wiki：里面包含了很多知识，基本可以按照这个入门。然后还提供循序渐进地刷leetcode题的平台，理论和实践配合，非常nice。配合CodeTop企业题库，完美!
  • 力扣leetcode：leetcode永远滴神，但是题目量太多，失去专一性，解答可以看题解或者Leetcode All in one
  • visualgo：数据结构和算法动态可视化网站，支持中文，支持对算法的详细说明和讲解，还可以查看算法的执行过程
  • LeetCodeAnimation：博主致力于将 LeetCode 上所有的题目都用动画的形式演示出来，计划用3到4年时间去完成它（哎，目前进度感人）
  • algorithm-visualizer：算法可视化网站，支持Java、JS、C++等多种语言，执行过程和速度可控制，还能暂停播放，在控制台也能看到算法执行输出的中间结果

数据库系统

学习路线

C++ 后台开发路线 2020

Web前端入门自学路线 2021

前端开发从入门到进阶完全指南

分类学习

前端

千古前端图文教程：不用去自己整理学习路线，只管学习就好，可以关注千古壹号博客园

Frontend Knowledge Structure：作者根据前端知识结构图，更新前端开发知识结构，非常细致

前端知识每日3+1/前端开发面试题：以面试题驱动学习，更新前端面试题

Offer-Javascript/剑指 offer–JavaScript 版/Javascript版 剑指offer：剑指offer编程题–JavaScript版本

Web开发入门：MDN的Web开发教程，为 Web 开发的纯粹初学者提供一切开始编写简单网站所需的知识

辅助网站

计算机书籍

脚本之家：专业的收藏整理多类脚本学习资料的个人网站

码农之家：给程序员和编程爱好者分享计算机编程电子书以及相关的学习资源和技术文章

How-to-be-a-good-programmer/CS-Book/free-programming-books-zh_CN:Github上比较好用的一些书籍分享项目

搬书匠：更新的是原版计算机类的书籍

其他学习

图解Git：图解git中的最常用命令

图说设计模式：使用图形和代码结合的方式来解析设计模式

SQL语句教程：网站列出及解释常用指令的语法，然后用例子让读者了解如何运用该指令

Learning Git Branching：在沙盒里执行相应的Git命令，还能看到每个命令的执行情况，学习Git最佳方式

Vim-adventures：游戏般学习Vim

常用工具

CodeIF：输入想命名的变量名，帮你选择英文变量名，避免无意义的中文变量名

开发者搜索：把百度搜索的内容过滤，筛选出和开发有关的答案

代码运行可视化：可以编写Python，Java，C，C++，JavaScript和Ruby代码，并逐步查看其执行可视化，了解计算机运行每行代码时会发生什么

‎JSON Editor Online：用于查看、编辑 JSON 并设置其格式的工具

CSS参考手册：提供CSS语法速查表，并提供chm版下载

Semi Design：抖音系的企业应用设计系统，设计语言、React 组件、主题等开箱即用的中后台解决方案
