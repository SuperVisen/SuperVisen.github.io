Hexo博客绑定域名

发表于2022-02-14|更新于2022-07-24|操作指南Hexo博客
|阅读量:

引子

​ &nbsp;👋这里是Visen.相信每个博主都希望自己的网站能够分享给大家看，而这一步就需要绑定域名。这里选择了阿里云购买域名，大家当然也可以通过腾讯云，大致过程都是一样的

本篇内容参考阿里云 域名帮助中心

购买域名

创建持有者信息

创建域名持有者信息模板，信息模板就是个人信息的模板，里面有实名信息，后续购买直接勾选该模板

进入万网（阿里云），点击右上角的登录，通过支付宝登录并完成注册

点击万网主页导航栏的控制台

在左上角选择域名，进入域名服务

点击信息模板，创建个人信息模板

填写基本信息，等待审核通过（两个小时左右）

挑选域名并购买

在万网首页搜索你想取的域名，我填入supervisen进行搜索（.com是世界顶级域名，.cn是中国顶级域名）

爱国情怀让我选择了.cn（也有部分因为没钱），加入清单，再进入域名清单进行结算

选择购买年限，选择所有者类型为个人并勾选刚才创建的信息模板，确定购买

绑定域名

获取博客IP地址

命令行输入ping 用户名.github.io（不要挂载一些代理，比如FastGithub）

进入万网的控制台，选择域名服务中的域名列表，选择刚购买的域名，进行解析

添加两个解析信息，使superisen.github.io与www.supervisen.cn链接起来（这里显示实名认证审核中，域名处于serverhold状态，此时无法使用）

创建CNAME文件

在本地博客根目录下的source文件夹里，创建一个名为CNAME的文件（注意要大写），写上购买的域名

进入GitHub中的博客仓库，绑定该域名

网站提交给搜索引擎

百度

百度中输入site:www.supervisen.cn，发现并未收录，点击提交网址

填入网址，如果出现您无权访问该页面，点击确定按钮返回首页，更换360极速浏览器再试试

回到主页百度搜索引擎平台,进入用户中心的站点管理

输入博客域名，勾选博客的站点属性

开始第三步的验证网站，有三种验证方法

  • 文件验证：下载验证文件放置在网站目录
  • HTML标签验证：在HTML代码中的head标签中插入元数据
  • CNAME验证：使用域名解析

这里选择第一种，点击下载好验证文件，放入博客的source目录下方（和CNAME文件放一起）

更改博客根目录下的_config.yml文件，选择跳过对这个HTML文件的“加工”

博客使用hexo clean && hexo g && hexo d上传到GitHub进行更新

点击确认进行验证文件正常访问，完成验证
